Comment renouveler (≈ 2 min)
  1. Va sur github.com/settings/personal-access-tokens, puis Generate new token.
  2. Donne un nom, mets une expiration (~1 an), Resource owner = ton compte.
  3. Repository access → Only select repositories → choisis dev-dashboard.
  4. Permissions → Repository permissions → Contents : Read and write.
  5. Clique Generate token, puis copie le jeton.
  6. Reviens ici, clique ⚠ Reconnecter (ou « Connecter GitHub ») et colle le jeton.

Synchroniser avec GitHub

Colle un jeton d'accès : tes favoris se sauvegarderont automatiquement sur GitHub et apparaîtront sur tous tes ordinateurs. Le jeton reste dans ce navigateur, jamais en ligne.

Créer le jeton → github.com/settings/personal-access-tokens/new
• Repository access : Only select repositories → dev-dashboard
• Permissions → Repository permissions → Contents : Read and write
