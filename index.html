Synchroniser avec GitHub

Colle un jeton d'accès : tes favoris se sauvegarderont automatiquement sur GitHub et apparaîtront sur tous tes ordinateurs. Le jeton reste dans ce navigateur, jamais en ligne.

Créer le jeton → github.com/settings/personal-access-tokens/new
• Repository access : Only select repositories → dev-dashboard
• Permissions → Repository permissions → Contents : Read and write
